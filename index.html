Matlock

Matlock is a Browser Extension that detects which Open Source libraries webpages are using.

Install for Chrome Install for Firefox

What is Matlock?

Matlock is our first attempt and building an extension (for Chrome and Firefox) which detects, and lists out, which Open Source libraries a webpage is using.

At the moment, this is limited to GitHub-hosted libraries.

How does it work?

Matlock works by running a series of tests on each page you navigate to, checking for specific variables, strings, headers or cookies.

The results of these tests tell Matlock whether a certain library is being used, and in some cases, which version.

Check out our FAQ for more information.
